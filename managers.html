Panya Managers¶

The following managers are contained in panya.managers.

PermittedManager¶

The permitted manager is used to limit queryset results based on object state and site eligibility, see ModelBase’s state and sites fields. The managers resulting queryset is determined by the following rules:

  1. Exclude all objects which have a state of unpublished.
  2. Only include those objects which have a state of published and a site set corresponding to the SITE_ID Django setting.
  3. If the setting STAGING is True, also include those objects which have a state of staging.

The permitted manager is accessible on the Panya ModelBase model’s permitted member.
